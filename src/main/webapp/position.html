position 기법(p280)

포지션은 웹문서에서 이미지나 글씨등을 원하는 위치에 자유롭게 둘때 사용합니다.

position 속성은 자주 사용하지만, 그 만큼 실수하기도 쉬운 부분이므로 꼼꼼하게 살펴보세요

    종류
  • static : 문서의 흐름에 맞게 배치(default), block은 block처럼 inline은 inline처럼
  • relative : 위치값(left,right,bottom,top)을 지정할 수 있다는 점을 빼고 static과 같다.
  • absolute : relative값을 사용한 상위요소를 기준으로 위치를 지정해 배치합니다.
  • fixed : 브라우저 창(=모니터)을 기준으로 위치를 지정해 배지합니다.
  • sticky : static+absolute 방식으로 동작하는 새로운 위치 종류

position 위치를 지정하는 속성

  1. left
  2. right
  3. bottom
  4. top
